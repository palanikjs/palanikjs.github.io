# palanikjs.github.io

Gold Command Center - Physical Gold Intelligence Dashboard

A comprehensive gold market analysis dashboard featuring:
- Live gold prices (24K, 22K, MCX futures, international)
- Key macro indicators (USD/INR, US yields, Sensex, crude oil, etc.)
- AI-powered market predictions and analysis
- Historical price charts and yearly returns
- Buy/sell history with outcomes
- Technical analysis and strategy recommendations
- Latest gold market news and key drivers
- India-specific factors and buying tips

Built with React and deployed on GitHub Pages.

**Live at:** https://palanikjs.github.io/
